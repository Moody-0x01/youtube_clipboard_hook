# Contributing

Thanks for wanting to contribute! A few pointers to get started:

1. Fork the repo and open a branch for your change.
2. Follow Rust formatting and linting rules. Run `cargo fmt` and `cargo clippy`.
3. Run the project locally while developing:

```sh
cargo run -- <download_path>
```

4. For changes to the Python UI, use the virtual environment in `app/` and run the server locally.
5. Open a pull request describing the change and include how to reproduce it.

Contact
- Open issues on GitHub with the `bug` or `feature` labels.
